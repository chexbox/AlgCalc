Algebra Calculator

This is a free open-scouce javascript calculator, capable of solving step-by-step algebra.
We hope to build this calculator for people to use for fun, homework, and business applications. We will use javascript to calculate the math and give output step-by-step. If we find someone who knows how to canvas, we will use it to add graphing functionality.
